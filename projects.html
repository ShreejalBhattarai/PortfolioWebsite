AI Room Brightness Controller

Raspberry Pi · Mac + Ollama (Mistral) · Python

View on GitHub
Python Raspberry Pi Flask PWM OpenCV Ollama / Mistral
  • Built an end-to-end system that controls a Raspberry Pi 3B+ LED over WiFi from a Mac, with a Flask REST server on the Pi driving GPIO18 via hardware PWM.
  • Implemented a manual mode that uses local speech recognition and a locally hosted Mistral LLM (via Ollama) to interpret natural-language commands like "dim it a little" into structured JSON brightness values.
  • Added an automatic mode that samples the Mac camera with OpenCV, computes grayscale room brightness, and inversely modulates the LED so darker rooms get a brighter LED.
